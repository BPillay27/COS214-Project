Detailed Description

Subclass of the Aggregate class. This class is a Concrete Aggregate which holds the rows of the 2D array.

bool PlantArea::addPlant ( Plant *  toAdd)
overridevirtual

Adds a plant.

Method to add a plant to the 2D array. This method begins by validating the given plant. Then, a row for this plant type is searched for. If found, p->addPlant(toAdd) is called, otherwise a new row is created for this new plant type and the plant is added there.

Returns
This method returns true if the plant was successfully added to the 2D array and false otherwise.
Parameters
toAddThe plant you wish to add to the 2D array.
