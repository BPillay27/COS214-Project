Detailed Description

The row of Plant objects in the 2d Array. This ConcreteAggregate holds the actual Plant references.

Adds a plant.

Method to add a plant to the array. Validates the plant and either places the plant in the appropriate row (If this plant type already had a row) or makes a new row and places the plant there (Plant type has no existing row).

Returns
