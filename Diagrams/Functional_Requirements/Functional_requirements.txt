The Functional requirements:
1) The plants should have a lifecycle and condition that needs to be taken cared of (overgrown, dehydration) which can be handled as a state.
2) The plant types should have different strategies to handle the different plant condition states.
3) There needs to be a single manner of keep track of the inventory and only one instance of an Inventory for better tracking.
4) The plants need some container to be easily stored in so new plants can be easily added.
5) New plants need to be created by when a type of plant is ordered for a restock.
6) The gardener need to be made aware of the different plants conditions so that they can attend to the plant.
7) The plant needs to make the gardener aware of when it is mature so that it can be sold.
8) The staff need to communicate with each to complete certain tasks through delegation to another that can do the task.
9) The customer needs to be able to be able to request things from the Salesman.
10) There needs to be some way for new customers to appear in the nursery.
11) The customer needs to be able to personalise there order with different arrangements of plants.
12) The customer's order personalisation needs to be able to cater for a plant being wrapped and have a decor pot in the order.
13) The needs to be a way for the customer to add a plant to the order and undo that action.
14) The system needs some way to show a payment being declined and allowing the customer to go their order again.
15) There needs to be an easy way to travel through areas of plants or storage containers of plants.
16) The plants need to show that they can die from age because they are an old plant and that they can die if their condition becomes really bad and the dying from both should behave the same way.

Non-functional requirements:
1) The singleton helps with reliability because it is a single instance therefore there are no inconsistencies between the instances of inventories.
2) The performance has an increase because the PlantArea improves finding a certain type of plant
3) There is maintainability because there will be documentation of the coed using deoxygen.
